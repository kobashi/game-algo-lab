## 1. 学習目標

1. **α**（MAX が確保した下限）と **β**（MIN が確保した上限）を説明できる  
2. **β カット**（MAX で v≥β）と **α カット**（MIN で v≤α）がいつ起きるか追える  
3. 素の Min-Max と同じ値を返しつつ、**読まない葉**があることを理解する  

---

## 2. アルゴリズム概要

```
AlphaBeta(n, α, β):
  if 葉: return score
  if MAX:
    v = -∞
    for c in children:
      v = max(v, AlphaBeta(c, α, β))
      if v ≥ β: return v   // β カット
      α = max(α, v)
    return v
  if MIN:
    v = +∞
    for c in children:
      v = min(v, AlphaBeta(c, α, β))
      if v ≤ α: return v   // α カット
      β = min(β, v)
    return v
```

根の呼び出し: `AlphaBeta(root, −∞, +∞)`

---

## 3. 初期木の意図

Min-Max と同型・同値の深さ3・葉12標準木（2026-07-19 改訂）。
根(MAX)→手 L/M/R(MIN)→応手(MAX)→葉 の3段構成にし、β カット・α カット・
内部節点ごとの深い刈り（枝丸ごと未評価）の3つを1本の木で見せる。

- 手 L: 応手 L1=max(5,3)=5 → L の β が 5 に。応手 L2 は最初の葉 L2a=7 ≥ β(5) で
  **β カット**（葉 L2b=9 は読まずに済む）→ L=min(5,7)=5  
- 手 M: 根の α が 5 になった後に開始。応手 M1=max(4,2)=4 ≤ α(5) で
  **α カット**（応手 M2 は節点ごと未評価＝葉2枚が丸ごと刈られる）  
- 手 R: 応手 R1=max(6,8)=8、応手 R2=max(7,4)=7（カットなし）→ R=min(8,7)=7  
- 根 = max(5, ≤4, 7) = 7（最善手 R）。評価した葉 9/12、カット回数2
  （β カット1・α カット1）
