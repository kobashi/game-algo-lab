# Game Algo Lab v1.0.0

GitHub Pages 向けゲームアルゴリズム可視化教材の初回リリースです。

## 実装済みデモ

- **BFS** — 幅優先探索（歩数最少、コストは参考表示）
- **DFS** — 深さ優先探索（再帰・コールスタック・バックトラック）
- **ダイクストラ** — 優先度 = g
- **最良優先探索** — 優先度 = h
- **A\*** — f = g + h

## 教材向け機能

- グリッド地図（`js/maps/*-map.js` をエディタで編集）
- データ構造の可視化（Queue / コールスタック / 親ポインタ list+tree など）
- C# 実装例（`samples/*.cs`、デモと 1:1 対応は不要）
- トピックメニュー（ゲーム木シリーズは準備中）

## 公開

静的サイトです。リポジトリ root を GitHub Pages の Source に設定してください。

python3 -m http.server 8080   # ローカル確認
