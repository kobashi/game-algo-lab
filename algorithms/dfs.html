深さ優先探索 (DFS)

再帰で深く潜り、行き止まりでバックトラック。 コールスタックを可視化。初期地図は js/maps/dfs-map.js。

このデモで学ぶこと・地図の記号
  • 1本の道をできるだけ深く進み、ダメなら戻る（バックトラック）
  • 再帰呼び出し = コールスタックにフレームが積まれる
  • return = スタックから外れる（戻って別の方向を試す）
  • 方向優先: 右 → 下 → 左 → 上（袋小路に入りやすい）
  • 最短経路は保証しない（BFS との対比）
