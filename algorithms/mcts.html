モンテカルロ木探索 (MCTS)

三目並べを題材に、探索木をプレイアウトで成長させる MCTS（UCT）を可視化する。 1 反復 = 選択 → 展開 → シミュレーション → 逆伝播。 腕の選び方は多腕バンディットの UCB1 と同じ形。 ルール・完全解・素の MC は tic-tac-toe デモと共有する。 設定: js/maps/mcts-config.js。

このデモで学ぶこと
  • 4相: 選択（UCB で降りる）→ 展開（未試行手を1つ木に足す）→ 乱択プレイアウト → 訪問数・勝率を根まで戻す
  • UCB1: \(\hat\mu + C\sqrt{\ln N_p / n_i}\)。各子 = バンディットの腕。未訪問は必ず一度選ぶ（∞）
  • 比較: 同一局面で MCTS・素のモンテカルロ推定・完全解（negamax）を並べ、反復 N を増やすと完全解の最善手に寄る様子を見る
  • 主シナリオ「ダブルリーチ罠」: 素の MC が低 N で外しやすい局面。MCTS は有望手に予算を寄せやすい
